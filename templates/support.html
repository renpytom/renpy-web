Ren'Py Support

Having trouble with Ren'Py? Try visiting the resources listed below.

Documentation

The documentation contains everything you need to know about how to use Ren'Py, including a quickstart guide and language primer.

Forum

The Ren'Py section of the Lemma Soft Forums is the best place to ask questions about Ren'Py.

IRC Channel

The IRC channel is another great place to talk about Ren'Py and creating visual novels. You can visit us on irc.freenode.net in #renpy. If you don't have an IRC client, there are a few Web based clients to let you get in on the fun!

Github

We use Github for project management, including code hosting and issue tracking.

Contact PyTom

PyTom is the creator of Ren'Py. You can contact him via email at pytom at bishoujo.us. While most messages will get a prompt reply, if you don't hear back in a few days your message might have been eaten by a spam filter. In that case please email renpytom at gmail.com, or PM him on the Lemma Soft Forums. (We apologize in advance for any inconvenience.)
